# Core Network Migration Plan

## Objective
Migrate from legacy ASR9010 to new ASR9906 core routers with MPLS Segment Routing.

## Timeline
- Week 1: Pre-migration preparation
- Week 2: Staging environment testing
- Week 3: Production migration
- Week 4: Post-migration optimization

## Success Criteria
- Zero unplanned downtime
- < 2 hours planned maintenance window
- All services operational post-migration
- BGP convergence < 60 seconds
